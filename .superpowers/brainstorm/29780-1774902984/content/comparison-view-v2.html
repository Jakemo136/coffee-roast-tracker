Design notes:
  • Transposed layout — roasts are rows, metrics are columns. Reads naturally like the dashboard roast table. Each row has a colored left border + dot matching the chart legend.
  • Same-bean comparison — the primary use case is comparing roasts of the same bean over time (dialing in a profile). Cross-bean comparison is allowed but not optimized for.
  • Entry points — "Compare all" on Bean Detail page compares all roasts for that bean. Dashboard multi-select + "Compare selected" for ad-hoc selection.
  • Bean column — uses short name for compactness. Shows the same bean name for same-bean comparisons, but differentiates when comparing across beans.
  • Rating — read-only StarRating. Unrated shows "—" instead of empty stars to keep the table clean.
