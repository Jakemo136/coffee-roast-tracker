Dashboard Behavior Notes

  • Row click → navigates to Roast Detail view
  • Rating stars → clickable inline (you chose B). Hover shows half-star targets. Click saves immediately via updateRoast mutation.
  • Flavor dots → colored circles from assigned flavor categories. Red dot = off-flavor present. Hover shows descriptor name.
  • Multi-select → checkbox appears on hover at row start. Select 2+ rows → "Compare selected" action bar slides up from bottom.
  • Upload button → opens quick-upload modal (you chose B)
  • Sort → click column headers. Default: Date descending.
  • Filter → bean dropdown + text search across bean name and notes.
  • Empty state → friendly message with upload CTA when no roasts exist.
